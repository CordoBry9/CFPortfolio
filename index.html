Bug Box

Bug Box

BugShot is a software defect tracking system. Commonly known as a bug tracker. BugShot is built with ASP.NET, Blazor, C# and SQL. A role based security system was implemented to create permission based roles. The roles of Admin , Project Manager, Developer and Submitter restrict what the user can see and do. The software allows for users to submit tickets and those tickets are tracked through status from Open to Complete.

The system was built with C# and .Net in 3 weeks. The front end was developed with HTML, Javascript and Bootstrap. The back-end utlizies Blazor and C#. The database was built with Postgre SQL and hosted on Railway. Please visit the site and try it out.
